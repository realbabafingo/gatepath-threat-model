# Security Policy

Gatepath aims to keep its infrastructure simple and transparent.

If you believe you have found a security issue affecting
the Gatepath infrastructure or documentation, please report
it responsibly.

## Reporting

Please send security-related reports to:

contact@gatepath.cc

Include:
- a clear description of the issue
- steps to reproduce, if applicable
- potential impact

## Scope

This repository contains documentation and example
configurations only.

Reports related to third-party software or Tor itself
should be directed to the respective upstream projects.

## Disclosure

Please allow reasonable time for investigation and response
before public disclosure.
